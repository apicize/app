Requests

Parameters Pane

This is pane to set parameters when testing your Request or Group. These parameters include:

  • Scenario: Name/value pairs injected in place of {{handlebars}} text
  • Authorization: Authorization sent with request
  • Certificate: Client certificate sent with request
  • Proxy: Proxy used to send request

These parameters are set hierchically, and default to the level above them, all the way "up" to the workbook's defaults. If you set any of these to a value, that value will be used regardless of any parent values set. If you set any of these to "Off", then no value will be used for that parameter, regardless of parent values.

If you want to run your tests by seeding external data, set that in workbook's defaults.
