# :icon[settings] Settings :toolbar

## Workbook Defaults :icon[defaults]

In this pane, you can configure the default parameters used for [Requests](help:workspace/requests):

* [Scenarios](help:workspace/scenarios)
* [Authorizations](help:workspace/authorizations)
* [Certificates](help:workspace/certificates)
* [Proxies](help:workspace/proxies)
* [Data Sets](help:workspace/data-sets)

These defaults can be overridden by any Request or Request Group.
