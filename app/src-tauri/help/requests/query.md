# :icon[request] Requests :toolbar

## Query String Pane :icon[query]

Use this pane to specify query string parameters to be appended to the URL specified in the [info pane](help:requests/info).  Each entry has a **parameter** and **value**.

To add a new query string parameter, click "ADD QUERY STRING PARAMETER".  To delete a query string parameter, click on the delete (trash can) icon.
